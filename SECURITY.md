# Security Policy

## Supported Versions

Currently this repository does not have a formal release versioning strategy. The main branch contains the latest updates. It is supported with the best effort for security issues.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests. 

If you believe you have found a security vulnerability in this project, please report it to us as soon as possible. Feel free to contact the maintainer directly. 

We take all issues seriously, and we will do our best to promptly address them.
